# Recipe_Web_App 
    Recipe web app using 
     - Spring Framework 5
         - Spring Data mongoDB
         - Spring Data Reactive Mongo
         - Spring webFLux
     - JUnit5 and Mockito for Testing
     - Project Lombok
     - QueryDSL for querying
     - CircleCi for continuous integration
     - Jacoco plugin for code coverage
